What To Include In A Support Request

Keep support requests structured so issues can be identified faster across multiple apps and release tracks.

  • App name Tell us exactly which app, game, or concept release your message is about.
  • Device details Include the Android version, device model, and a short summary of the issue.
  • Conversion or scanner issues When relevant, mention the file type, export format, or workflow step involved.
  • Privacy or deletion request If an app includes accounts later, include the identifier needed to locate the request.

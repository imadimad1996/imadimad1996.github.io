# imadimad1996.github.io

Public GitHub Pages website for **KestrelNebula Labs**, used as an Android developer website for:

- developer identity
- app support contact
- public privacy policy links
- Google Play verification readiness

## Live Site

Primary URL: `https://imadimad1996.github.io/`

## Project Structure

- `index.html`  
  Main homepage for the developer website.

- `privacy.html`  
  Public privacy policy page for website/support references.

- `404.html`  
  Custom not-found page for GitHub Pages.

- `robots.txt`  
  Search crawler rules.

- `sitemap.xml`  
  Sitemap for public pages.

## Purpose

This repo is intentionally simple and static. It is designed to provide a clean, professional, and easy-to-maintain public website that supports:

- clear developer branding
- visible support contact information
- stable public policy URLs
- fast loading and low maintenance

## Updating The Site

When editing this website, keep these details consistent with Google Play Console and public store information:

- developer name
- support email
- website URL
- privacy policy link
- app-related support or deletion instructions

If this developer account represents an organization, use official business contact details rather than temporary or personal placeholders.

## Deployment

This repository is a GitHub Pages user site. Changes pushed to the default branch are published at:

`https://imadimad1996.github.io/`

## Notes

- Keep public contact details accessible and up to date.
- Avoid placeholder links or outdated claims.
- If any app collects user data, make sure app-specific privacy disclosures remain accurate.
